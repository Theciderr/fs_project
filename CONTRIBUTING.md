# Contributing

Thanks for your interest in contributing to this project. A few guidelines to get started:

- Fork the repository and create a feature branch from `main`.
- Open a pull request describing the change and the rationale.
- Keep commits small and focused. Use conventional commit messages where possible.
- Run tests locally before opening a PR:
  - `cd backend && npm test`

Issues
- Please open issues for bugs or feature requests. Link related PRs to the issue.

License
- By contributing, you agree that your contributions will be licensed under the project's MIT license.
